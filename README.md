# 📱 High-Tech Product Specification Card

> Um componente de interface sofisticado para e-commerce de tecnologia, desenvolvido com HTML5 e CSS3, focado em clareza informativa e design minimalista.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir um **Cartão de Produto** (Product Card) otimizado para dispositivos eletrônicos, onde a densidade de informações técnicas precisa coexistir com um design limpo e moderno.

O componente apresenta uma visão equilibrada entre marketing (preço e Call-to-Action) e engenharia (especificações de hardware), garantindo que o usuário tenha todos os dados necessários para uma decisão de compra técnica.

## ✨ Destaques e Diferenciais Técnicos

* **Arquitetura de Layout Dual:** Divisão inteligente entre a área de branding/venda (`.right`) e a área de especificações técnicas (`.left`), otimizando o escaneamento visual.
* **Listagem Técnica Estruturada:** Uso de listas não ordenadas (`<ul>`) para apresentar dados complexos de hardware (Chipset, CPU, GPU) de forma organizada e legível.
* **Sistema de Avaliação Visual:** Integração com **Font Awesome (v5.15.2)** para exibir ratings por estrelas, utilizando classes sólidas e regulares para indicar a pontuação do produto.
* **Botão de Conversão Dinâmico:** Um Call-to-Action (CTA) robusto que combina o comando de compra com o valor do produto em uma única unidade visual, acelerando o fluxo de conversão.
* **Código Semântico e Limpo:** Uso de classes descritivas e organização profissional, mantendo a estrutura no `index.html` e o estilo isolado no diretório `css/style.css`.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de dados e componentes de interface.
* **CSS3:** Estilização de layout, tipografia e design responsivo.
* **Font Awesome (v5.15.2):** Biblioteca de ícones para suporte visual e prova social.
